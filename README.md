# Configurações do VSCode
Este repositório contém algumas configurações personalizadas para o Visual Studio
Code (VSCode). Aqui você encontrará um conjunto de ajustes que podem melhorar a sua
experiência de desenvolvimento no VSCode.

## Configurações disponíveis
**settings.json:** Este arquivo contém configurações do VSCode que afetam o ambiente
de trabalho, como o tema, as fontes, o espaçamento e as extensões recomendadas.

Sinta-se à vontade para personalizar essas configurações de acordo com suas
preferências.

**keybindings.json**: Neste arquivo, você encontrará configurações para mapear atalhos
de teclado personalizados. Isso permite que você execute ações específicas no VSCode
de maneira rápida e conveniente.

## Como utilizar

1. Faça o clone deste repositório para o diretório **'.vscode'** em sua pasta pessoal (ou
qualquer diretório de sua escolha) no seu sistema operacional.

2. Abra o VSCode e acesse as configurações clicando em **File > Preferences > Settings**.

3. Clique no ícone de três pontos no canto superior direito e selecione a opção
**Open Workspace Settings**. Isso abrirá o arquivo **'settings.json'** específico para
este projeto.

4. Copie o conteúdo do arquivo **'settings.json'** neste repositório e cole-o no
arquivo **'settings.json'** do seu VSCode.

5. Repita o passo 4 com o arquivo **'keybindings.json'**, copiando o conteúdo do arquivo
deste repositório e colando-o em seu arquivo **'keybindings.json'** do VSCode.

6. Salve as alterações e aproveite suas novas configurações personalizadas!

Fique à vontade para explorar e adaptar as configurações para atender às suas
necessidades de desenvolvimento. Esperamos que essas configurações facilitem seu
trabalho e tornem sua experiência no VSCode ainda mais agradável.
